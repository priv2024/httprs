`httprs` is a fast toolkit for probing hosts from stdin to stdout.
Probe hosts using custom regular expressions.

![preview](docs/preview.gif)

## Requirements

* OpenSSL with headers

## Usage

```shell
echo google.fr | ./httprs
```
